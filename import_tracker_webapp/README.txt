# 📦 Import Tracker — Setup Instructions

## Step 1: Install Flask (one time only)
Open VS Code terminal and run:
```
pip install flask
```

## Step 2: Run the app
In the terminal, go to this folder and run:
```
python app.py
```

## Step 3: Open on your LAPTOP
Open your browser (Chrome) and go to:
```
http://localhost:5000
```

## Step 4: Open on your PHONE 📱
1. Make sure your phone and laptop are on the SAME WiFi
2. On your laptop, find your IP address:
   - Windows: open terminal, type `ipconfig`, look for "IPv4 Address" (e.g. 192.168.1.5)
3. On your phone browser, go to:
   ```
   http://192.168.1.5:5000
   ```
   (replace with your actual IP)

That's it! Your data saves automatically to `data.csv` in this folder.
